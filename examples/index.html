jQuery insert plugin for MediumEditor

This is a demo of Hubrick's jQuery insert plugin for MediumEditor.

HINT: This whole website is an editable editor. You can click on text and start typing.

If you want to insert an image, Video or Hubrick Product, create a new empty paragraph and the plugin's button will appear on the left. From there it's pretty self-explanatory. Have fun.
